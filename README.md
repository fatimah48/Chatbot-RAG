# Chatbot-RAG

This project is AI Assistant Tool written by Python  that offers an intelligent chatbot interface to assist users with SAP-related queries(can be customized). The application leverages LangChain's Retrieval-Augmented Generation (RAG) approach, embedding vectors, and the Mistral model for advanced natural language processing.
